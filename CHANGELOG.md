# Changelog

All notable changes to Scene Map are documented in this file. Releases follow
[Semantic Versioning](https://semver.org/) and are managed by Release Please.

## 0.1.0 (2026-07-11)

- Added a local-first K-drama and film library with watch progress, ratings, and notes.
- Added AsianWiki title and full-cast importing with duplicate-safe relationship merging.
- Added cast history, interactive relationship graph, and shared-connection filtering.
- Added Docker Compose deployment, health checks, SQLite backup, and restore tooling.
